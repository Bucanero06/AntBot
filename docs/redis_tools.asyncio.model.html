redis_tools.asyncio.model

Exposes the Base Model class for creating custom asynchronous models.

This module contains the Model class which should be inherited when creating model’s for use in the asynchronous API of pydantic-redis.

class redis_tools.asyncio.model.Model[source]

Bases: AbstractModel

The Base class for all Asynchronous models.

Inherit this class when creating a new model. The new model should have _primary_key_field defined. Any interaction with redis is done through Model’s.

async classmethod insert(data: List[AbstractModel] | AbstractModel, life_span_seconds: float | None = None)[source]

Inserts a given record or list of records into the redis.

Can add a single record or multiple records into redis. The records must be instances of this class. i.e. a Book model can only insert Book instances.

Parameters:

  • data – a model instance or list of model instances to put into the redis store

  • life_span_seconds – the time-to-live in seconds of the records to be inserted. If not specified, it defaults to the Store’s life_span_seconds.

async classmethod select(columns: List[str] | None = None, ids: List[Any] | None = None, skip: int = 0, limit: int | None = None, **kwargs) → Model | Dict[str, Any][source]

Retrieves records of this Model from redis.

Retrieves the records for this Model from redis.

Parameters:

  • columns – the fields to return for each record

  • ids – the primary keys of the records to returns

  • skip – the number of records to skip. (default: 0)

  • limit – the maximum number of records to return

Returns:

By default, it returns all records that belong to current Model.

If ids are specified, it returns only records whose primary keys have been listed in ids.

If skip and limit are specified WITHOUT ids, a slice of all records are returned.

If limit and ids are specified, limit is ignored.

If columns are specified, a list of dictionaries containing only the fields specified in columns is returned. Otherwise, instances of the current Model are returned.

async classmethod update(_id: Any, data: Dict[str, Any], life_span_seconds: float | None = None)[source]

Updates the record whose primary key is _id.

Updates the record of this Model in redis whose primary key is equal to the _id provided. The record is partially updated from the data. If life_span_seconds is provided, it will also update the time-to-live of the record.

Parameters:

  • _id – the primary key of record to be updated.

  • data – the new changes

  • life_span_seconds – the new time-to-live for the record
